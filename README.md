## Getting Started :hammer_and_wrench:

Follow these simple steps to run the game locally:

### Prerequisites

Ensure you have **Node.js (v16+)** and **npm** or **yarn** installed.

### Installation

```bash
git clone https://github.com/Adil-km/TicTacToe-Game.git
cd TicTacToe-Game
npm install
```

### Run Locally

```bash
npm run dev
```

Then open the local URL (usually [http://localhost:5173](http://localhost:5173)) to start playing.
